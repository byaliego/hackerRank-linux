Problem Statement
Given two integers, X and Y, identify whether X<Y or X>Y or X=Y.

Input Format 
Two lines containing one integer each (X and Y, respectively).
Output Format 
Exactly one of the following lines: 
- X is less than Y 
- X is greater than Y 
- X is equal to Y
Sample Input 1
5  
2  
Sample Output 1
X is greater than Y  
Sample Input 2
2
2  
Sample Output 2
X is equal to Y   
Sample Input 3
2
3  
Sample Output 3
X is less than Y
